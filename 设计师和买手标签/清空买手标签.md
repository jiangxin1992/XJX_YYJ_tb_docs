清空买手标签
=======

**url**: __/statisticsTag/clearTagForBuyer
**method** : post  
**权限** : admin 
**调用方** : Web

### 请求参数

|     名称  	 |  类型   | 是否必填  |             说明                                                   |
|------------|--------|----------|-------------------------------------------------------------------|
| buyerId      | int | Y        | 买手id	                                                       |
### 请求示例
http://localhost:3334/__/statisticsTag/clearTagForBuyer
### 返回值示例 (接口返回值示例)
```json
{
  "data": null,
  "message": "成功",
  "status": 100
}
```
