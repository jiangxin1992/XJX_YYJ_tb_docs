判断订单是否为追单
=======

**url**: service/append/isAppend
**调用方**: web
**权限**: 设计师、零售商


### 请求参数

|       名称      |      类型      | 是否必填 |              说明             |
|-----------------|----------------|----------|-------------------------------|
| orderCode 	  | String         | Y        | 	订单号                    |

### 请求参数示例

```json
{
	"data":true,
	"message":"成功",
	"status":100
}
```
