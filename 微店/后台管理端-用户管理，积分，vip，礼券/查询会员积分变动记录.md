查询积分变动记录
=======

**url**: service/customer/listCustomerPointsLogs
**method** : get  
**权限** : RETAILER，BUYERSHOP  
**调用方** : Web

### 请求参数

|     名称  	 |  类型   | 是否必填  |             说明                                                   |
|------------|--------|----------|-------------------------------------------------------------------|
| customerId     | int    | Y        | 会员id                                                       |
| pageIndex     | int    | Y        | 页码                                                |
| pageSize     | int    | Y        | 每页条数                                              |

### 请求示例
http://localhost:3334/service/customer/listCustomerPointsLogs?customerId=10000023&pageIndex=1
