更新买手个人账号信息
==========

- **url**: /service/buyer/basicInfoUpdate
- **调用方**: web,app
- **权限**: 买手


### 请求参数

|    名称  		   |  类型     | 是否必存在 |                  说明                  |
|------------------|-----------|------------|----------------------------------------|
| name			   | string    | 否         |零售商名称 							 |
| phone			   | string    | 否         |零售商电话, 需要包含国际码							 |
