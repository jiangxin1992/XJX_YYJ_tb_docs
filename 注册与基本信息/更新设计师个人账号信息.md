更新买手个人账号信息
==========

- **url**: /service/designer/basicInfoUpdate
- **调用方**: web,app
- **权限**: 买手


### 请求参数

|    名称  		   |  类型     | 是否必存在 |                  说明                  |
|------------------|-----------|------------|----------------------------------------|
| userName			   | string    | 否         |名称 							 |
| phone			   | string    | 否         |电话, 需要包含国际码							 |
