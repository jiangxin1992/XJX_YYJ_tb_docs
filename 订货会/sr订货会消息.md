sr订货会消息
=======

## 查看

- **url**: /service/sr_appointment/msg
- **method** : get
- **权限** : showroom,showroom_sub
- **调用方** : web/ios

### 请求参数

无

### 返回值说明

true表示有消息；否则无



## 清除

- **url**: /service/sr_appointment/msg
- **method** : delete
- **权限** : showroom,showroom_sub
- **调用方** : web/ios

### 请求参数

无

### 返回值说明

可忽略
