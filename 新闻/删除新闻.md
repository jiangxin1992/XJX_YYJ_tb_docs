删除新闻
=======

**url**: service/news/delete
**调用方**: web,app
**权限**: 后台


### 请求参数
|    名称   |  类型  | 是否必填   |                  说明                  |
|-----------|--------|------------|----------------------------------------|
| id        | int    | 是         |新闻id								   |

### 请求参数示例

```json
{
{
	"data":1,
	"message":"成功",
	"status":100
}
}
```
