新增菜单
=======

- **url**: /__/menu
- **method**: POST
- 权限: admin

### 请求参数

#### 主菜单

|  名称 | 是否必填 |  类型  |        说明        |
|-------|----------|--------|--------------------|
| name  | Y        | string | 名称               |
| order | Y        | number | 排序值(越小越靠前) |
| icon     | Y        | string |                    |

#### 二级菜单

|   名称   | 是否必填 |  类型  |        说明        |
|----------|----------|--------|--------------------|
| name     | Y        | string | 名称               |
| order    | Y        | number | 排序值(越小越靠前) |
| parentId | Y        | string | 父菜单id           |

| url      | Y        | string | 路径               |


### 返回值说明

返回的是新增菜单的id
