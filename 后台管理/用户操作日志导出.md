用户操作日志导出
=======

- **url**: /__/data/user_ops/export
- **权限**: admin
- **method**: GET


### 请求参数

|   名称   |  类型  | 是否必填 |              说明             |
|----------|--------|----------|-------------------------------|
| token    | string | Y        | 权限token                     |
| userType | number | Y        | 账号类型，0 品牌；1买手       |
| start    |        | Y        | 开始日期: 2018-05-21 00:00:00 |
| end      |        | Y        | 结束日期: 2018-05-21 23:59:59 |
| status   | number | N        | 为空表示所有                  |

### 返回值说明

返回的是一个二进制文件流
