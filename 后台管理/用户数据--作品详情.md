作品详情
====

- **url**: /__/data/brand/series
- **method**: GET

### 请求参数

|    名称    |  类型  | 是否必填 |   说明   |
|------------|--------|----------|----------|
| designerId | number | Y        | 设计师id |

### 返回值说明

|      名称     |  类型  |     说明     |
|---------------|--------|--------------|
| seriesName    | string | 系列名称     |
| seriesId      | number | 系列id       |
| seriesCreated | number | 系列创建日期 |
| type          | string | 类型         |
| styleCount    | number | 款式数量     |
