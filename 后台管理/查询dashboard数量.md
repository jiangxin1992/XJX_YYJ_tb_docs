查询dashboard数量
=======

**url**: /__/dashboard/amount

### 请求参数
| 名称 |  类型  | 是否必填 |  说明  |
|------|--------|----------|--------|
| name | string | 是       | 类型,见下面类型说明 |

类型说明:
- designer: 处于正常状态的设计师数量
- series: 处于发布状态的系列数量
- series: 处于发布状态的系列数量
- style: 处于正常状态的款式数量
- buyer: 处于正常状态的买手数量
- order: 总的订单数量

### 返回值说明

|  名称  | 类型 |     说明     |
|--------|------|--------------|
| amount | int  | 数量         |
| raw    | int  | 未过滤的数量 |

### 返回值示例

```
{
  "data":{
    "amount":13
  },
  "message":"成功",
  "status":100
}
```
