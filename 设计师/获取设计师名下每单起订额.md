设计师名下每单起订额
=======

### 获取设计师个人信息

- **url**: /service/preference/orderUnitPrice
- **method**: `post`

#### 请求参数

|    名称    | 类型 | 必填 |               说明               |
|------------|------|------|----------------------------------|
| designerId | int  | N    | 买手调用此接口需要传入，其它不必 |
| curType    | int  | Y    | 货币类型                         |

####  响应参数说明

| 名称 |  类型  |    说明    |
|------|--------|------------|
| data | number | 每单起订额 |

##### 样例

```json
{
  "data": 3000,
  "message": "成功",
  "status": 100
}
```
