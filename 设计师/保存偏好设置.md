保存记好设置
=======

注意: 新增/修改和删除都使用此接口，提交数据时，提交该偏好类型下的所有json数据

- **url**: /service/designer/prefer/save
- **method**: `post`

#### 请求参数

| 名称 |     类型    | 必填 |           说明           |
|------|-------------|------|--------------------------|
| type | int         | Y    | [偏好类型](/common/常量.md) |
| data | json string | Y    | json数据                 |

####  响应参数说明

略
