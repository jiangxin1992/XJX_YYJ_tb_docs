获取openId
=======

**url**: user/weChat/getOpenId

### 请求参数
|     名称    | 类型 | 是否必填 |                                    说明                                   |
|-------------|------|----------|---------------------------------------------------------------------------|
| code | String  | Y      | 用户code 用户获取openId

### 返回示例

{
  "data": {
	openId:wxa12y4ty3t4444
   }
    ,
  "message": "成功",
  "status": 100
}
