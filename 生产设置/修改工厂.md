修改工厂
=======

**url**: /service/order/updateManufacturer
**method**: post
**权限**: 设计师、销售代表


### 请求参数

|     名称     |  类型  | 是否必填 |   说明   |
|--------------|--------|----------|----------|
| name         | string | Y        | 工厂名称 |
| address      | string | N        | 工厂地址 |
| introduction | string | N        | 工厂介绍 |
| id           | number | Y        | 工厂         |

### 返回字段说明

略
