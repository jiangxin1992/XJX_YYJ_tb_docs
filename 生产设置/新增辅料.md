新增辅料
=======

**url**: /service/auxiliary
**method**: post
**权限**: 设计师/销售代表


### 请求参数

|    名称   |  类型  | 是否必填 |    说明    |
|-----------|--------|----------|------------|
| supplier  | string | Y        | 供应商名称 |
| img       | string | N        | 图片       |
| name      | string | Y        | 面料名称   |
| code      | string | Y        | 编码       |
| unitPrice | number | Y        | 单价       |
| comment   | string | N        | 备注       |
| year      | number | Y        | 年份       |
| season    | string | Y        | 季节       |

### 返回字段说明

返回的是新建的面料的id
