保存购物车税率
=======

**url**: /service/preference/taxRate
**method**: post
**权限**: all
**调用方**: Web

### 请求参数
|   名称  |  类型  | 是否必填 |    说明    |
|---------|--------|----------|------------|
| taxRate | number | Y        | 购物车税率 |

### 返回字段说明

忽略

#### 返回字段说明

```
{
  "data": 1,
  "message": "成功",
  "status": 100
}
```
