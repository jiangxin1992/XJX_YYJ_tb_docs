类目表

重要字段:

1. id: 类目id
2. people_type: 一级类目
3. suit_type: 二级类目

### desc

```
+-------------+-------------+------+-----+---------+----------------+
| Field       | Type        | Null | Key | Default | Extra          |
+-------------+-------------+------+-----+---------+----------------+
| id          | int(11)     | NO   | PRI | NULL    | auto_increment |
| people_type | int(11)     | YES  |     | NULL    |                |
| suit_type   | int(11)     | YES  |     | NULL    |                |
| name        | varchar(50) | YES  |     | NULL    |                |
| name_en     | varchar(50) | YES  |     | NULL    |                |
| sort_id     | int(11)     | YES  |     | NULL    |                |
+-------------+-------------+------+-----+---------+----------------+
```
