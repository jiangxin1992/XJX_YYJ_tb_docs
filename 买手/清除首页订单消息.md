清除首页订单消息
========

- **url**: /service/notify/index_msg/clear
- **method**: post
- **平台**: Web/Pad
- **权限**: 零售商


### 参数

|  名称  |  类型  |        说明        |
|--------|--------|--------------------|
| status | number | 清除哪种状态的消息 |

### status 对照表

| 值 |        含义        |
|----|--------------------|
|  4 | 待协商             |
|  5 | 待签合同           |
|  6 | 合同已签(即待发货) |
|  8 | 已发货(即待收货)   |

#### 返回值示例

```json
{
    "data":null,
    "message":"成功",
    "status":100
}
```
