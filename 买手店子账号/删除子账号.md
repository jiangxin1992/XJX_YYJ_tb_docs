删除子账号.md
==============

##### 删除子账号的接口

**url**:/service/buyerSubaccount/delete
**权限** : RETAILER 
**method** : post   
**调用方** : Web

### 请求参数
|    名称                    | 类型    | 是否必填 |           说明           |
|---------------------------|---------|-------|--------------------------|
| id        				| int     | Y     | 子账户id                  |

### 返回值示例 (接口返回值示例)

```json
{
    "data": [
        {
            message: "操作成功"
            status: 100
            
        }
    ],
    "message": "成功",
    "status": 100
}
```
