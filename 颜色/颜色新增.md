颜色新增
=======

**url**: /service/designer/color/add
**method**: post
**权限**: 设计师
**调用方**: Web

### 请求参数

|     名称    |  类型  | 是否必填 |             说明             |
|-------------|--------|----------|------------------------------|
| name        | string | 是       | 颜色名称                     |
| value       | string | 是       | 色值，只允许以#或http(s)开头 |
| description | string | Y        | 描述/备注                    |

### 返回字段说明 (接口返回值说明)

| 名称 | 类型 |     说明     |
|------|------|--------------|
| data | int  | 新增颜色的id |

### 返回值示例 (接口返回值示例)

```json
{
  "data": 3,
  "message": "成功",
  "status": 100
}
```
