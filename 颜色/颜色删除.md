颜色删除
=======

**url**: /service/designer/color/delete
**method**: post
**权限**: 设计师
**调用方**: Web

### 请求参数

| 名称 | 类型 | 是否必填 |   说明   |
|------|------|----------|----------|
| id   | int  | Y        | 颜色的id |

### 返回字段说明

| 名称 | 类型 |    说明    |
|------|------|------------|
| data | int  | 受影响行数 |

### 返回值示例

```json
{
  "data": 1,
  "message": "成功",
  "status": 100
}
```
