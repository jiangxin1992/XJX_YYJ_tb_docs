创建支付宝付款对象
=======

**url**: /service/account/operate
**method** : post  
**权限** : all  
**调用方** : Web,APP

### 请求参数

|     名称      |  类型  | 是否必填 |             说明                                |
|---------------|--------|----------|------------------------------------------------ |
| operate	    | String | Y        | 支付宝账号   				0：关闭  1：开启	  |


### 返回值示例 (接口返回值示例)

```json
{
	"data":null,
	"message":"成功",
	"status":100
}
```
