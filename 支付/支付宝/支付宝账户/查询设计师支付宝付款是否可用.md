创建支付宝付款对象
=======

**url**: /service/account/isAvailable
**method** : post  
**权限** : 需买手登录
**调用方** : Web,APP

### 返回参数说明
|     名称       |    类型   | 是否必填 |      说明      	 				  |
|----------------|-----------|----------|-------------------------------------|
| designerId     | String    | 是       | 									  |


### 返回值示例 (接口返回值示例)

```json
{
	"data":false,
	"message":"未添加支付宝账户",
	"status":100
}
```
